# Memento Params - Parameter Visualizer

自由自在に項目を定義して、複数のデータをレーダーチャートで比較可視化できる「モバイルファースト」なデータ分析ツールです。

## 特徴

- **自由な項目定義**: 比較したい軸（項目）の名称、最小値、最大値、単位を自由に設定可能。
- **モバイルファースト設計**: Memento Database 風の UI を採用し、スマートフォンからでも快適にデータの入力・閲覧が可能。
- **プライバシー重視**: サーバーにデータを送信せず、全てのデータをブラウザ内のローカルデータベース（RxDB + Dexie）に保存。
- **画像シェア機能**: 作成したレーダーチャートを美しいカード形式の画像としてダウンロードし、SNS 等で簡単にシェア可能。
- **テンプレート機能**: 「ビジネススキル」「サッカー選手能力」「RPGキャラ」「ワイン評価」など、すぐに使えるプリセットを搭載。

## 活用シーン

- **自己分析**: スキルセットや健康状態の推移を可視化。
- **趣味の比較**: ワイン、コーヒー、ゲームキャラの性能比較など。
- **仕事**: プロジェクトごとのリソース配分やチームのスキルバランスの確認。

## 技術スタック

- **Frontend**: Next.js 15 (App Router), React, Tailwind CSS
- **Database**: RxDB (Offline-first / Local Storage)
- **Visual**: Apache ECharts
- **Components**: shadcn/ui
- **Icons**: Lucide React

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。
