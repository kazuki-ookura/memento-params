# データ構造定義 (Dynamic Schema)

## Field (項目定義)

ユーザーが定義した「軸」のリスト。

| プロパティ | 型     | 説明                               |
| :--------- | :----- | :--------------------------------- |
| id         | string | 識別子                             |
| name       | string | 項目名（営業力、攻撃力など）       |
| min        | number | 最小値（チャートの表示範囲に利用） |
| max        | number | 最大値（チャートの表示範囲に利用） |
| unit       | string | 単位（cm, pt, % など）             |

## Profile (プロファイル)

ユーザーが比較対象（自分、キャラクターAなど）ごとに作成する単位。

| プロパティ | 型     | 説明                         |
| :--------- | :----- | :--------------------------- |
| id         | string | 識別子                       |
| name       | string | プロファイル名               |
| color      | string | グラフ表示時の色 (HEXコード) |

## Entry (記録エントリー)

各プロファイルに紐づく特定の時点（年齢、Lvなど）の実データ。

| プロパティ | 型      | 説明                                                |
| :--------- | :------ | :-------------------------------------------------- |
| id         | string  | 識別子                                              |
| profileId  | string  | Profileへの参照                                     |
| label      | string  | ラベル（13y, Lv10, 2024/02など）                    |
| values     | object  | `{ "field-id": 85 }` 形式のデータ (fieldId -> 数値) |
| visible    | boolean | グラフ表示のON/OFF                                  |

## 備考

- **ローカルファースト**: 初期実装では全て localStorage に保存するが、将来的に JSON/CSV のインポート・エクスポートをサポートする。
