# 製品定義書: Universal Parameter Visualizer & Database

## 1. サービス概要

「貯める」だけでなく「比べる」に特化した、軽量・汎用データベース。ユーザーが定義した任意のパラメータを自由に登録し、1タップで複数の記録をレーダーチャートなどで視覚的に比較できるツール。

## 2. ターゲット

- 既存のDBアプリやスプレッドシートの「グラフ化・比較」のステップが面倒だと感じているユーザー
- 自分の成長記録やキャラ設定など、複数の数値を「セット」として視覚的に捉えたいユーザー
- ログイン不要で、今すぐ手元の数値を構造化して管理したいユーザー

## 3. MVP (必須機能)

- **ダイナミック・スキーマ管理**: 項目（軸）の自由な追加・編集、および「テンプレート（ビジネススキル、ゲームキャラ、ワイン評価等）」による即時開始。
- **クイック比較エンジン**:
  - 最大5つのデータをチェックするだけで、即座にレーダーチャートで重なりを可視化。
  - スマホでの操作を第一に考えた、一括データ入力グリッド。
- **データ永続化**: ブラウザのローカルストレージを活用した、ログイン不要のセキュアな保存。

## 4. データ戦略

- **ユーザー入力データ**: 自由度の高い入力フォーム。
- **拡張性**: JSON/CSVによるインポート・エクスポートによる、自作スプレッドシートとの連携（将来の拡張）。
