# UI/UX 仕様書

## 1. 比較インターフェース

- **年齢層セレクター**: サイドバーまたはヘッダーで、比較したい年齢層（18-19, 20-24, ... 70+ 等）を最大5つチェック可能にする。
- **比較モード**:
  - **Grid View**: チャートを横に並べて個別に確認。
  - **Overlay View**: 一つのチャート内に全選択データを重ねて表示（色の透過度で判別）。

## 2. データビジュアライゼーション詳細

- **企画・実行相関図**:
  - X軸: 企画力, Y軸: 実行力。
  - 各ドットはプロジェクトまたはチームのパフォーマンス。
- **スキルスコア分布**:
  - 1〜5点評価等のスコアリング軸でのヒストグラム。
